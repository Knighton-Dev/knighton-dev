# Knighton Dev LLC - Public Assets

This directory contains static assets that will be served from the root of your domain.

## Usage

Files in this directory are served at the root path. For example:
- `/public/logo.png` → `https://knighton.dev/logo.png`
- `/public/favicon.ico` → `https://knighton.dev/favicon.ico`

## Recommended Assets

Consider adding:
- `favicon.ico` - Browser tab icon
- `logo.png` - Company logo
- `og-image.png` - Open Graph image for social sharing (1200x630px)
- `robots.txt` - Search engine crawling instructions
- `sitemap.xml` - Site structure for search engines

## Optimization

- Compress images before adding them
- Use appropriate formats (WebP for photos, SVG for logos)
- Keep file sizes small for better performance
