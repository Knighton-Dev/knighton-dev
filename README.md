# Knighton Dev LLC

Marketing website for Knighton Dev LLC - Platform, DevOps, and Site Reliability Engineering services.

**Live Site**: [knighton.dev](https://www.knighton.dev)

## Tech Stack

- Next.js 15 with TypeScript
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit http://localhost:3000

## Build

```bash
npm run build
npm start
```
